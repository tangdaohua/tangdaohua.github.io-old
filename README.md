## 忆秦娥·娄山关
毛泽东
1935年

西风烈，
长空雁叫霜晨月。
霜晨月，
马蹄声碎，喇叭声咽。

雄关漫道真如铁，而今迈步从头越。
从头越，
苍山如海，残阳如血。

## 忆秦娥·箫声咽
唐.李白

箫声咽，秦娥梦断秦楼月。秦楼月，年年柳色，灞陵伤别。

乐游原上清秋节，咸阳古道音尘绝。音尘绝，西风残照，汉家陵阙。
